3. If using ISO files, see [INSTALL_SACD_EXTRACT.md](INSTALL_SACD_EXTRACT.md)

### For Developers
1. Review [WORKING_DIRECTORY_IMPLEMENTATION.md](WORKING_DIRECTORY_IMPLEMENTATION.md) for architecture details
